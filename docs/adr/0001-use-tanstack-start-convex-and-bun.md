# Use TanStack Start, Convex, and Bun

Aviously is moving away from the copied Next.js portfolio template and toward TanStack Start on Bun, with Convex as the default backend for experiential inputs and later dynamic content. This keeps the site close to React, file-based routing, server functions, and live data while giving Avi a playground for custom interactions instead of a static portfolio shell.

**Considered Options**:
Staying on Next.js would be more familiar and deployment-simple, but it would keep the project psychologically tied to the copied template. A plain Vite app would be simpler, but TanStack Start gives SSR, routing, and server-side affordances that fit a personal site with experiments, intake flows, and future content.
